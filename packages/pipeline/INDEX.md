## üìã Command Quick Reference

```bash
# View help
python run-esia-pipeline.py --help

# Run all steps (default)
python run-esia-pipeline.py

# Run specific steps
python run-esia-pipeline.py --steps 1          # Extract
python run-esia-pipeline.py --steps 3          # Analyze
python run-esia-pipeline.py --steps 1,3        # Extract & Analyze

# Debug mode
python run-esia-pipeline.py --verbose

python run-esia-pipeline.py --version
```
