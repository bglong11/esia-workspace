Pipeline Architecture

The ESIA pipeline processes documents through three main stages, each with specialized AI-powered analysis.
1
PDF Chunking (Docling)
Converts PDF documents into semantic chunks using Docling hybrid chunking. Tracks page-level progress and preserves document structure for downstream processing.
2
Fact Extraction
Extracts domain-specific facts using archetype-based pattern matching with LLM integration. Features intelligent caching with 37-80% hit rate for improved performance.
3
Analysis & Reporting
Analyzes extracted facts for consistency, compliance issues, and generates comprehensive reports with actionable insights and recommendations.
